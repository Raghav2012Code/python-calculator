# 🔢 Python Calculator

A simple command-line calculator built with Python. Supports basic arithmetic, square root, and exponentiation.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division (with zero division error handling)
- √ Square root (handles negative numbers gracefully)
- 🔋 Exponentiation (power)
- 🔁 Menu loop until user exits
- 🚫 Input validation (division by zero, negative square root)

## 🛠️ How to run

1. Make sure Python 3.6+ is installed.
2. Clone the repository or download `main.py`.
3. Open terminal in the project folder.
4. Run the command:
   ```bash
   python main.py
